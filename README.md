# robosys2022

# ファイル名
plus

# 導入方法
```
git clone https://github.com/nanaminakayama/robosys2022.git
```

# プログラムの説明
これは与えられた数字のプラスとマイナスを絶対値で取り、全てを足し合わせるプログラムです。

# 必要なソフトウェア
* Python:[3.7,3.8,3.9,3.10]

# テスト環境
Ubuntu18.04LTS

# 実行例
```
seq 10 | ./plus
55
seq -10 1 10 |./plus
110
```
