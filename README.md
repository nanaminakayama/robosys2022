# robosys202x
ロボットシステム

File name:
plus

Description:
これは与えられた数字のプラスとマイナスを絶対値で取り、全てを足し合わせるプログラムです。


Requirement:
Ubuntu18.04LTS
Python 3.7,3.8,3.9,3.10

behavior example:
seq 10 | ./plus
55
seq -10 1 10 |./plus
110
